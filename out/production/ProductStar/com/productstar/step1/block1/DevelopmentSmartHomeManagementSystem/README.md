# Система управления умным домом (Smart Home)

Учебный проект по Java, реализующий базовую систему управления умными устройствами в доме.

## 🏠 Функциональность

- Абстрактный класс `SmartDevice` с общими полями (имя, включено, комната).
- Устройства: умная лампа (`SmartLight`), термостат (`SmartThermostat`), телевизор (`SmartTV`).
- Интерфейс `Controllable` для регулировки (яркость, температура, громкость).
- Класс `SmartHome` с массивом устройств (добавление, удаление, включение/выключение всех).
- Вложенный статический класс `HomeStats` для статистики использования.
- Перечисление `RoomType` (гостиная, спальня, кухня, ванная, кабинет).

## 🚀 Как запустить

1. Открой проект в IntelliJ IDEA.
2. Запусти класс `Main` в пакете `com.productstar.step1.block1.DevelopmentSmartHomeManagementSystem`.

## 📊 Пример вывода

При запуске программа создаёт три устройства, включает их, регулирует параметры, затем выключает все и показывает статистику.
