# Симулятор кофейни "JavaBeans"

Программа для обучения бариста: симуляция работы кофейни с управлением меню и заказами.

## 🚀 Как запустить

1. Открой проект в IntelliJ IDEA.
2. Найди и запусти класс `Main` в папке `CoffeeShopSimulator`.

## 🧠 Что делает программа

*   Принимает заказы на кофе, чай и выпечку из меню.
*   Учитывает размер напитка (Small, Medium, Large).
*   Выводит чек и общую статистику продаж.

## 📚 Стек технологий

*   Java 17
*   IntelliJ IDEA
*   Git

## 📁 Структура проекта

*   `MenuItem` — абстрактный класс для всех позиций меню.
*   `Coffee`, `Tea`, `Pastry` — конкретные типы продуктов.
*   `CoffeeShop` — основной класс для управления заказами.
*   `CoffeeShopStats` — сбор статистики продаж.
