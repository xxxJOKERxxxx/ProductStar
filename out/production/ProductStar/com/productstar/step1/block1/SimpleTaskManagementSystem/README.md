# TaskMaster — система управления задачами

Учебный проект для отработки ООП на Java. Реализует базовую систему управления проектами и задачами.

## Функциональность

- Создание задач с названием, описанием, сроком выполнения и статусом.
- Управление статусами: TODO, IN_PROGRESS, DONE.
- Проекты содержат коллекции задач.
- Сортировка задач по разным критериям (название, дата и т.д.).
- Статистика по проектам: общее количество задач, сколько выполнено.
- Менеджер проектов для работы с несколькими проектами.

## Как запустить

1. Открой проект в IntelliJ IDEA.
2. Запусти класс `Main` в пакете `com.productstar.step1.block1.TaskManagementSystem`.

## Структура проекта

- `TaskStatus` — перечисление статусов задачи.
- `Task` — класс задачи.
- `Manageable` — интерфейс с методами assign, start, complete.
- `Project` — класс проекта (реализует Manageable, содержит задачи).
- `TaskComparator` — вложенный класс для сортировки задач.
- `ProjectStats` — статический класс для статистики.
- `TaskManager` — класс для управления проектами.
