# Круг (Circle) на Java

Учебный проект: класс `Circle` с инкапсуляцией, проверкой радиуса и вычислением площади.  
Включает тестовый класс `TestCircle` для демонстрации работы.

## Возможности
- Создание круга с заданным радиусом
- Геттер и сеттер для радиуса
- Проверка: радиус должен быть > 0, иначе `IllegalArgumentException`
- Вычисление площади: `π * r²` (через `Math.PI`)
- Тестирование в консоли с обработкой исключений

## Как запустить
```bash
javac Circle.java TestCircle.java
java TestCircle
